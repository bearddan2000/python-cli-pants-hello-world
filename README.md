# python-cli-pants-hello-world

## Description
POC for pants build tool for python.

## Tech stack
- python
- pants

## Docker stack
- pantsbuild/centos7:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
